# Quizzes

This is an example project for me to learn [React](https://facebook.github.io/react/). It uses [Redux](http://rackt.org/redux/docs/basics/UsageWithReact.html), [Material-UI](http://callemall.github.io/material-ui/), [Firebase](https://www.firebase.com), and is built on top of the [Material-UI Example Webpack Project](https://github.com/callemall/material-ui/tree/master/examples/webpack-example).

Build and dev tools used include: [Webpack](https://webpack.github.io), [Babel ES6](https://babeljs.io), [ESLint](http://eslint.org), [Velocity React](https://github.com/twitter-fabric/velocity-react), [Redux DevTools](https://github.com/gaearon/redux-devtools), and [React Hot Loader](http://gaearon.github.io/react-hot-loader/),

I gave up trying to handle asynchronous calls to read the Firebase data. Instead I followed [this guy's method] (https://github.com/erikras/react-redux-universal-hot-example/issues/252) of importing Firebase and listening for changes.

Animation taken from [this guy's method] (https://github.com/twitter-fabric/velocity-react/blob/master/demo/components/loading-crossfade-component.jsx)
